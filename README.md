## Webpack2 Demo

Messing with Webpack2, ES6, PostCSS...

### Install & Launch Component Demo
1. `git clone https://github.com/davidgarsan/webpack2-demo.git --branch feature/templates --single-branch`
2. `yarn install && npm run start` or `npm install && npm run start`
3. `npm run start`
4. `open http://localhost:9000`
